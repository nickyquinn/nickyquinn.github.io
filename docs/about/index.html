About me

I'm Nick - a software developer based in the UK.

I got my start on BBC Micro and Acorn computers, building websites in the days of tables, spacer gifs, and Internet Explorer 5. Since then I've worked across a lot of stacks, and these days I'm mostly found building web-based applications with .NET and React.

This site is where I write about things I find interesting — mostly software, occasionally not.
