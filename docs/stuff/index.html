Stuff

Things I've liked, purchased, or found useful.

  • 'Domain Driven Design: Tackling Complexity in the Heart of Software' by Eric Evans The definitive resource for anyone interested in modeling software to match the domain.
  • Darknet Diaries by Jack Rhysider True stories from the dark side of the internet.
  • The Missing Cryptoqueen (BBC) The story of Dr Ruja Ignatova, who persuaded millions to invest in her cryptocurrency OneCoin, then vanished.
  • Smashing Security by Graham Cluley Weekly cybersecurity news and discussion, without the jargon.
  • Oh My Posh A prompt theme engine for any shell.
  • Home Assistant Open source home automation that puts local control and privacy first.
  • Privacy Guides Tools and guides to protect your privacy online.
  • Model F Keyboards Reproduction IBM Model F keyboards, built to the original buckling spring standard.
  • Webaim accessibility evaluator tool A must-have for meeting web accessibility guidelines.
  • CSS Tricks For frontend, probably the site I refer to most.
